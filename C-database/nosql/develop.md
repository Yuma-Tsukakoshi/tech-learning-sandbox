# NoSQLデータベース開発ガイド

## 開発環境のセットアップ

### 必要条件
- Docker Desktop
- MongoDB 6.0以上
- MongoDB Compass（GUIツール）

### セットアップ手順
1. MongoDBの起動
```bash
docker-compose up -d
```

2. 接続確認
```bash
mongosh
```
